GET /api/restaurants

                            
Search for restaurants near a location

                            
                            

                                
cURL Example

                                
curl "http://localhost:5001/api/restaurants?lat=3.140853&lng=101.693207&keyword=pizza"

                            


                            

                                
Parameters:

                                
    
                                    
  • lat (float, required): Latitude (-90 to 90)
    • 
                                    
  • lng (float, required): Longitude (-180 to 180)
    • 
                                    
  • keyword (string, optional): Search keyword. Default: "food"
    • 
                                    
  • limit (int, optional): Max results (1-100). Default: 32
    • 
                                    
  • use_cache (bool, optional): Use cached results. Default: true
    • 
                                

                            

                        


                        

                            
POST /api/refresh-token

                            
Refresh authentication tokens using Playwright

                            
                            

                                
cURL Example

                                
curl -X POST "http://localhost:5001/api/refresh-token" \
  -H "Content-Type: application/json" \
  -d '{"lat": 3.140853, "lng": 101.693207}'
